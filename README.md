# 🚀 LawyerDex

🌐 Live Demo: https://your-live-url.com
📂 Showcase Repo: https://github.com/ayrsync/lawyerdex-showcase

A modern SaaS platform for lawyer listings, lead generation, and subscription-based services.

---

## 📌 Overview

LawyerDex is a full-stack application designed to help users find legal professionals while enabling lawyers to manage their profiles, leads, and subscriptions through a powerful dashboard.

---

## ✨ Key Features

- 🔐 Authentication & Role-Based Access (Admin / Lawyer / User)
- 📊 Dashboard with analytics and insights
- 💳 Subscription system (monthly, yearly plans)
- 📥 Lead management system
- 🌍 Location-based lawyer listings
- ⭐ Reviews & ratings
- ⚡ SEO-optimized listing pages

---

## 🛠 Tech Stack

### Frontend

- Next.js 14+
- TypeScript
- Tailwind CSS
- ShadCN UI

### Backend

- PHP Laravel
- MYSQL

### Other Tools

- Stripe (payments)
- Google Maps API (location)
- JWT Authentication
